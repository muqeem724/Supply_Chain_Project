# Step 5: Solutions Guide - Supply Chain Management System

## Introduction
This guide provides the SQL solutions for the practice questions in Step 4. Use these to check your work and learn more efficient ways to write queries.

## Section A: Level 1 (Basic Queries)

### 1. Read the data from all the tables
```sql
SELECT * FROM supplier;
SELECT * FROM product;
SELECT * FROM orderitem;
SELECT * FROM orders;
SELECT * FROM customer;
```

### 2. Find the country-wise count of customers
```sql
SELECT country, COUNT(id) AS count_cust 
FROM customer 
GROUP BY country 
ORDER BY count_cust DESC;
```

### 3. Display the products that are not discontinued
```sql
SELECT * FROM product 
WHERE isdiscontinued = '1';
```

### 4. Display the list of companies along with the product name that they are supplying
```sql
SELECT s.companyname, p.productname 
FROM supplier s 
JOIN product p ON s.id = p.supplierid;
```

### 5. Display customer information about who stays in 'Mexico'
```sql
SELECT * FROM customer 
WHERE country = 'Mexico';
```

### 6. Display the costliest item that is ordered by the customer
```sql
SELECT c.id AS customer_id, p.productname, p.unitprice 
FROM customer c 
JOIN orders o ON c.id = o.customerid
JOIN orderitem oi ON o.id = oi.orderid
JOIN product p ON p.id = oi.productid
WHERE (c.id, p.unitprice) IN (
    SELECT c.id, MAX(p.unitprice) 
    FROM product p
    JOIN orderitem oi ON oi.productid = p.id
    JOIN orders o ON o.id = oi.orderid
    JOIN customer c ON o.customerid = c.id
    GROUP BY c.id
);
```

### 7. Display supplier ID who owns the highest number of products
```sql
WITH max_p AS (
    SELECT s.id, COUNT(p.id) AS cnt_prod
    FROM supplier s 
    JOIN product p ON p.supplierid = s.id
    GROUP BY s.id
)
SELECT * FROM max_p 
WHERE cnt_prod IN (SELECT MAX(cnt_prod) FROM max_p);
```

### 8. Display month-wise and year-wise counts of the orders placed
```sql
SELECT YEAR(orderdate) AS year, MONTH(orderdate) AS month, COUNT(id) AS order_count 
FROM orders
GROUP BY YEAR(orderdate), MONTH(orderdate) 
ORDER BY year ASC, month ASC;
```

### 9. Which country has the maximum number of suppliers?
```sql
WITH max_c AS (
    SELECT country, COUNT(id) OVER(PARTITION BY country) AS count
    FROM supplier
)
SELECT DISTINCT * FROM max_c 
WHERE count IN (SELECT MAX(count) FROM max_c);
```

### 10. Which customers did not place any orders?
```sql
SELECT * FROM customer
WHERE id NOT IN (SELECT customerid FROM orders);
```

## Section B: Level 2 (Intermediate Queries)

### 1. Arrange the Product ID and Name based on the high demand by the customer
```sql
SELECT p.id AS product_id, p.productname, SUM(oi.quantity) AS total_quantity
FROM orderitem oi 
JOIN product p ON p.id = oi.productid
GROUP BY p.id, p.productname 
ORDER BY total_quantity DESC;
```

### 2. Display the total number of orders delivered every year
```sql
SELECT YEAR(orderdate) AS year, COUNT(id) AS total_orders 
FROM orders 
GROUP BY YEAR(orderdate);
```

### 3. Calculate year-wise total revenue
```sql
SELECT YEAR(orderdate) AS year, SUM(totalamount) AS total_revenue 
FROM orders 
GROUP BY YEAR(orderdate);
```

### 4. Display the customer details whose order amount is maximum including his past orders
```sql
SELECT c.*, SUM(totalamount) AS total_order_amount 
FROM customer c
JOIN orders o ON o.customerid = c.id
GROUP BY c.id 
ORDER BY SUM(totalamount) DESC 
LIMIT 1;
```

### 5. Display the total amount ordered by each customer from high to low
```sql
SELECT c.*, SUM(totalamount) AS total_order_amount 
FROM customer c
JOIN orders o ON o.customerid = c.id
GROUP BY c.id 
ORDER BY SUM(totalamount) DESC;
```

### 6. List the current and previous order dates for each customer
```sql
SELECT c.id AS customerid, c.firstname, o.orderdate
FROM customer c 
JOIN orders o ON o.customerid = c.id
ORDER BY customerid, orderdate;
```

### 7. Find out the top 3 suppliers in terms of revenue generated by their products
```sql
SELECT s.id AS supplierid, s.contactname, SUM(quantity * oi.unitprice) AS totalrevenue
FROM supplier s
JOIN product p ON s.id = p.supplierid
JOIN orderitem oi ON oi.productid = p.id
JOIN orders o ON o.id = oi.orderid
GROUP BY s.id, s.contactname 
ORDER BY totalrevenue DESC 
LIMIT 3;
```

### 8. Display the latest order date of all the customers with customer details
```sql
SELECT c.*, MAX(orderdate) AS latest_order_date
FROM customer c
JOIN orders o ON o.CustomerId = c.id
GROUP BY c.id;
```

### 9. Display the product name and supplier name for each order
```sql
SELECT oi.orderid, p.productname, s.contactname AS suppliername
FROM supplier s 
JOIN product p ON p.supplierid = s.id
JOIN orderitem oi ON oi.productid = p.id
JOIN orders o ON o.id = oi.orderid;
```

## Section C: Level 3 (Advanced Queries)

### 1. Fetch the customer details who ordered more than 10 products in a single order
```sql
WITH product_ordered AS (
    SELECT DISTINCT c.*, COUNT(o.id) OVER(PARTITION BY c.id) AS count
    FROM customer c 
    JOIN orders o ON o.customerid = c.id
)
SELECT * FROM product_ordered 
WHERE count > 10 
ORDER BY count DESC;
```

### 2. Display all the product details with the ordered quantity size as 1
```sql
SELECT p.* 
FROM product p 
JOIN orderitem oi ON oi.productid = p.id 
WHERE quantity = 1;
```

### 3. Display the companies that supply products whose cost is above 100
```sql
SELECT DISTINCT s.* 
FROM supplier s 
JOIN product p ON p.SupplierId = s.id
JOIN orderitem oi ON oi.productid = p.id 
WHERE oi.unitprice > 100;
```

### 4. Display the customer list who belongs to the same city and country arranged country-wise
```sql
SELECT DISTINCT c1.id, c1.firstname, c2.FirstName, c1.city, c1.country
FROM customer c1 
JOIN customer c2 ON (c1.city, c1.country) = (c2.city, c2.country)
WHERE c1.id <> c2.id
ORDER BY c1.country;
```

## Section D: Level 4 (Expert Queries)

### 1. Calculate the total amount saved in each order and rank by savings
```sql
SELECT oi.orderid, SUM((p.unitprice - oi.unitprice) * oi.quantity) AS total_amount_saved
FROM product p 
JOIN orderitem oi ON p.id = oi.productid
GROUP BY oi.orderid 
ORDER BY total_amount_saved DESC;
```

### 2. Market Research for Mr. Kavin
```sql
-- a. High-demand products
SELECT p.productname, SUM(oi.quantity) AS total_demand
FROM product p
JOIN orderitem oi ON p.id = oi.productid

```
